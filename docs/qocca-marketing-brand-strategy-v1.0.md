## 9. クマちゃん (Claude Code) の運用ルール

今後の実装判断で:

1. **UI 文言追加時**: NG 語彙チェック → 推奨ワードで言い換え
2. **SNS テンプレ追加時**: NG 語彙 0件を確認してから INSERT
3. **ブログ生成時**: gpt-4o prompt に「NG 語彙集 + 推奨ワード」を組み込む (Phase 3a 修正検討)
4. **HomePage / About 等の主要 UI 改修時**: `Live with pets.` 反映を必ず提案
5. **広告クリエイティブ (Meta) 時**: 「絶対に伝える 3つ」を必ず含む
