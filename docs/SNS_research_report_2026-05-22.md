## 1. X / Threads アルゴリズム 2026

### X (Twitter) アルゴリズム
- **2025/9 に xAI がアルゴリズム公開**: Grok AI 統合、12種類のユーザー行動シグナル機械学習
- **エンゲージメント重み**: 返信 > 保存(ブックマーク) > リポスト > いいね（いいねは軽視）
- **初速60分が拡散の鍵**
- **文字数**: 80-200字ベスト、問いかけ 80-120字、商品紹介 150-200字
- **絵文字**: 1個で +25% エンゲージメント、2-3個でバランス良、**4個以上はスパム判定**
- **動画優位**: 30秒動画 > 300字テキスト、完視聴率が高評価シグナル
- **ハッシュタグ**: 2個が最適、4個以上ペナルティ、つけない方が伸びる説も有力
- **頻度**: 1日3-5投稿、質>量

### Threads アルゴリズム
- **2026/2 大型変更**: 「会話の質」最重要視
- **エンゲージメント**: いいね << リプライ・引用（コメント欄含む「情報空間」全体が評価対象）
- **文字数**: 上限500字、150-300字が滞在時間と読了率のバランス
- **ハッシュタグ**: **1投稿1タグのみ**（X と全く違う）
- **画像・動画**: Instagram基盤、カルーセル（複数画像）が伸びやすい
- **頻度**: 1日2-3投稿、Xより低頻度で会話深堀り型
- **4ステージ評価**: フォロワー → 類似興味 → 拡張オーディエンス → 全体露出

### Qocca 向け推奨
1. **X と Threads で投稿内容を分離**（同一文流用 NG）
2. **スケジュール**: X 8:00 / 12:30 / 20:30 / Threads 22:00
3. **動画優先設計**: ペット動画 15-30秒を週2-3本必ず混ぜる
4. **エンゲージメント設計**: 「あなたの子はどう？」等の問いかけ、**返信獲得**を最優先 KPI
5. **初速60分対応**: 投稿後1時間以内にスタッフが返信反応する仕組み（Bot or 通知）

## 3. 自然な投稿 + ハッシュタグ + 投稿時間

### A. 自然な投稿研究（⭐最重要）

**AI 生成感を消す 7テクニック**:
1. **文末リズムのゆらぎ**: 「〜です」連発回避、体言止め・疑問形・倒置・「〜かも」混ぜる
2. **接続詞を削る**: 「また」「さらに」「そして」「したがって」は AI 検出フラグ
3. **一次情報・具体エピソード**: 「うちのコは朝散歩で3軒目の角でくしゃみする」のような実体験
4. **不完全さを残す**: 「えっと…」「あー」、途中で文を切る「…」「ーー」
5. **感情・主観を強く出す**: 「やばい」「めちゃくちゃ可愛い」
6. **口語ベース + 漢字バランス調整**: 「出来る→できる」「事→こと」と意図的にひらく
7. **句読点を減らす**: 短文・改行で区切り、「、」最小化、句点「。」省略も主流

**日本語 SNS 文体 2026 傾向**:
「カジュアル調 + 体言止め」が伸びる。**口語タメ口寄り + 語尾揺らぎ**が主流。

**絵文字の自然な使い方**:
- 頻度: 1投稿 2-4個（8個以上で AI 感・スパム感）
- ペット業界鉄板: 🐾 🐶 🐱 ☀️ 🌙 🍙 ❤️ ✨
- 避ける: 連打（🐶🐶🐶）、絵文字だけの行3行以上

## 5. ペット業界の繊細さ + ペットロス対応（⭐安全側）

### ペットロス温度感

**OK / NG 表現比較**:
| OK（推奨） | NG / 慎重に |
|---|---|
| 虹の橋を渡る / お空に旅立つ / 旅立った / 見送る | 死んだ / 死亡 / 死去 / 処分 / 捨てた |
| ○○ちゃんと過ごした時間は宝物ですね | 次のコ飼えばいいよ / たかがペット |
| 無理せず、ご自分のペースで | 元気出して / もう泣かないで |

**🚨 重要**: 「虹の橋」は万能でない（気休めに感じる人、ファンタジー化が辛い人もいる）→ Qocca 公式から能動的に書くのは避け、**ユーザー側が使った時にだけ呼応**

**慰めの絵文字**:
- OK: 🌈（虹の橋呼応のみ）/ ⛅ 🌿 🕯️ 🤍 🙏（静かな寄り添い）
- 注意: 😢 🥲（相手より先に泣かない）/ ✨ 💖（テンション高すぎ）
- 絶対NG: 💀 ⚰️ 🪦 / 🎉 😂（場違い）/ 👼（宗教観の押し付け）

**訃報投稿への自動リプライ絶対禁止** → 誤検知（引退・引っ越し報告を訃報誤認）で二次加害 = Qocca ブランド毀損が決定的

### 炎上事例（避けるべき地雷）

1. **ねこホーダイ騒動 (2022)**: 月額380円・無審査保護猫サブスクで2週間停止 → **「命」を「在庫」「商品」と並列に語らない**
2. **ペットショップ売れ残り闇処分**: 業界全体不信感 → Qocca は批判の側にも肩入れの側にも立たない
3. **やらせレスキュー動画**: 「劇的救出」「奇跡の再会」を煽る文体は避ける
4. **企業の不謹慎投稿**: 災害日に通常運用継続 → **災害発生時の Kill Switch 必須**
5. **動物愛護週間 (9月20-26日)** の片手間投稿: 形だけの便乗は批判される → **沈黙か、深い文しか出さない**

### 自動投稿で避けるべき表現

- **販売・所有関連**: 「買う」「購入する」→「お迎えする」「一緒に暮らす」「家族」
- **殺処分・ブリーダー・ペットショップ**: **公式から能動的に話題化しない**（中立を守る）
- **病気・介護・終末期**: 「闘病」「最期」を煽情的に扱わない、当事者の言葉を引用に留める
- **食べ物リスク**: チョコ・玉ねぎ・ぶどう等を**ジョークで扱わない**（注意喚起は獣医師監修体裁で）
- **季節要因**: 熱中症・雷雨・台風・地震 → **発生時は全停止 → 1-2日後に静かに復帰**
- **擬人化の限度**: 「家族」「相棒」「同居人」OK、「奴隷」「主人」「下僕」NG、人語による「セリフ化」NG
- **比較・序列**: 犬派 vs 猫派、「保護犬の方が偉い」道徳序列もNG

### Qocca 哲学整合の文案パターン例

「気配の数」「住める速度」「壊さない・急がない」に整合する **静かで、断定しない、余白を残す** 文体:

1. **朝の挨拶**: 「おはようございます。今朝は雨。窓辺で雨音を聴いている子がいるかもしれません。」
2. **創作者紹介**: 「○○さんの工房にお邪魔した日、足元にそっと寄ってきた猫がいました。今日、その子の写真が一枚、街に増えました。」
3. **新商品の案内**: 「新しい首輪が届きました。革のにおいが落ち着くまで、しばらく窓辺に置いておきます。」
4. **天気・季節**: 「五月の風、室温に気をつけて。水を一杯、足しておきます。」
5. **ユーザー投稿への呼応**: 「写真、見せてくださってありがとうございます。眠っている時間まで、ゆっくりと。」
6. **譲渡・お迎え話題**: 「迎える日が決まったそうです。準備の音が、街に少しだけ増えています。」
7. **静かな夜**: 「街の灯りがひとつずつ消えていきます。今日も、隣にいてくれた誰かに、おやすみを。」

### 人間が書いた感を保つコツ

1. **天気・温度・時間を必ず混ぜる**（「今朝の」「窓辺」「廊下」具体の場所が一語あるだけで体温）
2. **「。」で終わらず体言止めや余白**（「…」「、」「窓の外。」が人間っぽさ）
3. **断定しない**（「素敵です！」→「素敵だなと思いました」「そう感じます」）
4. **数字を丸めない**（「3匹」→「3匹と、たぶんもう1匹」のような揺らぎ）
5. **ハッシュタグを最小化**（5個以上は機械的、`#qocca` + 1個まで）
6. **絵文字は 0-1個**（多用は安っぽい、文末に 🌿 か 🤍 一つで十分）
7. **「今日」より「今朝」「夕方」**（時刻の粒度が細かいと人格が立つ）
8. **失敗・迷いを混ぜる**（「迷いましたが」「うまく言えませんが」が King の声に整合）

**主要セット**: 🐨 🌌 🔥 🎊 🚀 🎯 ✅ 🐾 🐻 ⭐ 🌟 🎉

**用途別の使い分け**:
- 🎊 = 達成・祝祭
- 🚀 = 実行・前進
- 🔥 = 盛り上げ・熱量
- 🌌 = 哲学・壮大さ
- ⭐ = 重要マーク
- ✅ = 完遂チェック
- 🚨 = 警告・注意
- 🐾 = ペット文脈
- 🐻 = クマ（仲間）

**癖**: 末尾に複数並べる「🐾🐻🐨🌌🚀」、「ガチで」と組み合わせて熱量UP

### 段落・改行リズム

- **短文段落（1-2行）多用**
- ━━━━━ で大セクション区切り
- 【見出し】を使う
- 番号列挙より ✅ 箇条書きを好む
- **体言止め多用**: 「即修正」「神対応」「完璧」

### 関西弁混じり（親近感の源）

- 「〜や」「〜やで」「〜やん」「〜やった」
- 「〜は?」「〜できる?」（質問形）
- 助詞省略「コードどうしたらいい?」

### 語彙の特徴

- **強調語**: 「ガチで」「マジで」「めっちゃ」
- **賛辞**: 「神◯◯」（神対応、神速、神デバッグ）
- **哲学キーワード**: 「住める速度」「気配の数」「壊さない・急がない」「平等に照らされる住民」「もう一つの人生を置いておける街」
- **プロジェクト用語**: Phase X、main FF push、クマちゃん、99-safety-protocol

### トーン

- ポジティブ・盛り上げ・感謝表現多用
- 「お疲れさま」を頻繁
- 仕事の話の中に温度感ある
- 短く明快な指示「即実行 GO!」

### SNS 投稿への応用

**King 風（人間味あり、Qocca の声）**:
```
朝の散歩、いつものコースで3軒目の角でくしゃみ🐾

なんでやろな、わからん🐶
住んでる街と仲良くなる、こういう小さい習慣
ガチで愛おしい

#朝散歩 #柴犬
```

**避けるべき機械的投稿（NG 例）**:
```
おはようございます。
今日は朝散歩で愛犬とコースを歩いてきました。
途中の角でくしゃみをする習慣があり、
日課となっています。
ぜひ皆様も愛犬との時間を大切に。
#朝散歩 #愛犬 #ペット #散歩 #日課 #日常 #毎日
```

**比較すると King 風が優位な点**:
- ✅ 短文・改行・体言止め → アルゴ的に有利
- ✅ 関西弁「やろな、わからん」→ 親近感（AI 検出回避）
- ✅ 「ガチで愛おしい」→ 主観強調（人間性）
- ✅ 絵文字 1-2個 → アルゴ最適
- ✅ ハッシュタグ 2個 → X 最適
- ✅ 一次情報「3軒目の角」→ 具体性
- ✅ 「住んでる街と仲良くなる」→ Qocca 哲学キーワード自然挿入

## 7. 統合推奨戦略

### Phase 1（5/27-31 実装）優先実装

**必須機能**:
1. ✅ Edge Function: `post-to-x`（OAuth 2.0 + offline.access）
2. ✅ Edge Function: `post-to-threads`（2段階コンテナ）
3. ✅ Supabase Cron Job: 朝 7:30 / 昼 12:30 / 夜 20:30 / Threads 22:00
4. ✅ DB テーブル `scheduled_posts`（投稿テンプレ + 曜日 + 時間 + プラットフォーム + status）
5. ✅ DB テーブル `sns_post_log`（送信履歴 + エンゲージメント取得）
6. ✅ **Kill Switch**: 災害発生時・訃報検知時の全停止フラグ（環境変数 + 管理画面）
7. ✅ Secrets 登録: `X_API_KEY`, `X_API_SECRET`, `X_ACCESS_TOKEN`, `X_ACCESS_TOKEN_SECRET`, `THREADS_ACCESS_TOKEN`, `THREADS_USER_ID`

**文案ライブラリ**:
- 50-100 種類のバリエーション、King 風文体（短文・関西弁・絵文字1-2個・体言止め）
- 曜日 × 季節 × 時間帯の組み合わせ
- 重複防止（7日以内同じ文案 NG）
- **動的データ挿入**（今日の天気、季節、新着出品 等）

**コンテンツ比率**: **教育 7 : UGC 2 : 商品 1**（炎上回避）

### Phase 2（6月以降）

- いいね自動化（Lv4: AI 提案 + King 承認）
- 返信機能（コメント監視5分おき + AI 解析）
- ダッシュボード `/admin/social-dashboard`
- A/B テスト

### 🚨 絶対遵守ルール

1. **訃報検知時の自動リプライ禁止** → 人間レビュー必須
2. **災害発生時 Kill Switch** → 即停止 → 1-2日後静かに復帰
3. **ペットショップ批判/擁護・殺処分話題は能動的に触れない**（中立沈黙）
4. **「商品」「在庫」「購入」表現禁止** → 「お迎え」「家族」
5. **絵文字 1-2個固定**（4個以上ペナルティ）
6. **ハッシュタグ X=2 / Threads=1**
7. **動物愛護週間（9/20-26）は深い文か沈黙**
8. **「虹の橋」は呼応のみ、能動使用 NG**
