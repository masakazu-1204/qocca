# 経済層 弁護士確認リスト (依頼書 #133 Phase A1)

日付: 2026/6/6
適用範囲: ポイント/アイテム/ギフト/クラファン特典/手数料計算
作成根拠: King 確定済の経済層設計思想 + 資金決済法対応

## 確認項目

### 1. 🔴 ポイント = 行動報酬のみ・購入不可 (= 前払式支払手段に非該当)

- ✅ ポイント購入機能は **永続的に実装しない** (King 明示指示 / 2026/6/6 承認)
- 設計上の根拠:
  - 資金決済法 第3条「前払式支払手段」 = 対価として発行され、商品/役務と引換可能な番号・記号
  - 本サービスのポイントは「ブログ執筆 / ギャラリー投稿 / コミュニティ参加 / 施設レビュー / 取引完了」等の **行動報酬** としてのみ発行
  - 対価 (現金) の払込なしに発行される = 「対価として発行」に非該当
  - → 「前払式支払手段」に該当しない = 供託義務・届出義務・財務局検査対象 すべて非該当
- 弁護士確認: ◯◯先生に「上記スキームが前払式に該当しないこと」の意見書を依頼

### 2. 🟠 アイテム購入時の 10pt 還元 = 「無償付与 (おまけ)」 (Stripe 規約改訂時の再確認項目)

- ✅ アイテム購入 (Stripe Checkout) 時に 10pt をユーザーに付与
- 法的整理:
  - 10pt は「物品購入の対価」として発行するのではなく、購入行為への「無償の感謝の付与」 (景品扱い)
  - 景品表示法上の「景品類」(取引付随性のある物品提供) には該当する可能性あり
  - ただし上限規制 (景品表示法 第4条) には 100pt = 100円相当 として収まる ( ≪ 取引価額の 10%)
- 🔄 **Stripe 規約改訂時に再確認すべき項目**: Stripe の「ロイヤルティポイント / リワード」関連条項が更新された場合、本スキームが規約抵触しないか確認 (年1回見直し推奨)
- 弁護士確認: 上記の景品扱いで OK か / Stripe 規約への適合確認

### 3. 🟢 アイテム直接購入のみ採用 (ポイント購入なし)

- 各アイテム ¥200-¥2,000 (税込) を Stripe Checkout で都度購入
- 自社販売モード: `qocca_fee=0` / `shipping_fee=0` / 物品 → ダウンロード型 (デジタル装飾) または 即時消化
- 即時消化のため「未使用残高」概念がなく前払式支払手段に非該当

### 4. 🟢 クラファン特典 「手数料 3%」 の長期確約 (景品表示法 適合)

- リターン: 「事業が存続する限り 創業期クリエイターは手数料 3%」 (`founding_creator_fee_rate=3`)
- 「事業が存続する限り」表現で適法 (「永久 / 永遠 / 無期限」NG = 依頼書 #101 既対応)
- 期間条件: welcome 期間中 (~7/31) は 0% (= 一般ユーザーと同等) / 8/1 以降に 3% (= 創業者特典発動)
- **「特典保持者がキャンペーンで損する逆転を絶対に作らない」(King 指示):** 最有利率を採用 = welcome 中 0% / 通常期 3% / standard 10%
- 弁護士確認: クラファン CAMPFIRE リターン文言と本実装の整合性

### 5. 🟢 ギフト = 受け取り拒否デフォルト ON / 公開ランキングなし

- `gift_settings.accepting_gifts` デフォルト = true (受け取り可)
- `gift_settings.hide_gift_log_publicly` デフォルト = true (公開ログ無効)
- 送信者の匿名化はオプション (デフォルト = 記名 / King 修正指示)
- 設計思想: 「返礼圧は絶対作らない」 / 「ギフトランキング作らない」
- 弁護士確認: 贈与税の発生有無 (現状 pt = 数百円相当 → 110万円/年の贈与税基礎控除内で大幅余裕 / 個人間贈与に該当する場合の補足説明)

### 6. 🟢 既存 home_atmosphere (5プリセット) は無料維持

- 既存住民 18名の体験を 1ミリも下げない保証 (King 指示)
- items テーブル統合後も「無料カテゴリ」として全住民が引き続き使える
- 弁護士確認: 既存ユーザーへの不利益変更に該当しないこと

### 7. 🟢 創業期特典の表示文言確認

- App.tsx L7877 既存表記: 「事業が存続する限り手数料 3% (通常10% → 創業特典)」
- 「永久 / 永遠 / 無期限」非含有 (依頼書 #101 適用済)
- 弁護士確認: 「事業が存続する限り」の解釈余地と免責文言の必要性
